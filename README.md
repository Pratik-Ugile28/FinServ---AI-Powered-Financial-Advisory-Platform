# FinServ – AI-Powered Financial Advisory Platform

## 🚀 Overview

FinServ is a full-stack AI-powered financial advisory platform designed to provide intelligent financial planning, investment tracking, and personalized financial recommendations using modern AI technologies.

The platform leverages **Retrieval-Augmented Generation (RAG)**, **Large Language Models (LLMs)**, and **Vector Databases** to deliver context-aware financial assistance and smart decision-making support.

---

## ✨ Features

### 🤖 AI Financial Advisor
- RAG-powered chatbot for intelligent financial guidance
- Context-aware responses using Groq LLM and LangChain
- Personalized financial recommendations

### 🔍 Semantic Search with Vector Database
- Pinecone vector database integration
- Embedding-based semantic retrieval
- Fast and accurate financial document search

### 👤 User Authentication & Security
- JWT-based authentication
- Password hashing using bcryptjs

### 💰 Financial Management

### 📊 Risk Assessment & Analytics

### 🌐 Dynamic Data Collection

# 🧠 AI Workflow

1. User sends a query to the chatbot
2. Query embeddings are generated
3. Pinecone retrieves relevant financial context
4. LangChain constructs the final prompt
5. Groq LLM generates context-aware responses
6. Response is returned to the user
